# kotlin-minesweeper

# 1단계 - 지뢰 찾기(그리기)

## 기능 요구사항

지뢰 찾기를 변형한 프로그램을 구현한다.
- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- 지뢰는 눈에 잘 띄는 것으로 표기한다.
- 지뢰는 가급적 랜덤에 가깝게 배치한다.

## 기능 구현

- CellType
  - [ ] 지뢰 타입을 지정 가능하다.
- Position
  - [ ] x, y 좌표를 가진다.
- Cells
  - [ ] 위치와 지뢰 타입을 가진다.
  - [ ] 지뢰 갯수 지정 가능하다.
  - [ ] 지뢰를 랜덤으로 배치한다.
