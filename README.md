### STEP 1

<summary>접기/펼치기</summary>
<div markdown="1">

#### [요구 사항 분석]

- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- 지뢰는 눈에 잘 띄는 것으로 표기한다.
- 지뢰는 가급적 랜덤에 가깝게 배치한다.

#### [기능 목록]

- [x] 높이를 입력 받는다.
- [x] 너비를 입력 받는다.
- [x] 지뢰 개수를 입력받는다.
- [x] 2차 배열에 입력된 지뢰 개수만큼 랜덤으로 지뢰(💣)를 생성한다.
- [x] 2차 배열을 출력한다.
