# kotlin-minesweeper

## 지뢰 찾기

### 기능 요구사항

- [x] 높이와 너비, 지뢰 개수를 입력받는다.
- [x] 지뢰는 `*`, 빈칸은 `C`로 표시한다.
- [x] 지뢰판을 생성한다.
    - [x] 지뢰 갯수는 높이와 너비의 곱보다 작아야 한다.
    - [x] 지뢰는 지뢰판 범위를 넘어갈 수 없다.
    - [x] 지뢰는 중복될 수 없다.
    - [x] 지뢰는 랜덤으로 배치한다.
    - [x] 크기가 0인 지뢰판은 생성할 수 없다.
- [x] 특정 좌표의 지뢰 여부를 판단한다.
- [ ] 지뢰 주변 8방향에 숫자를 표기한다.
