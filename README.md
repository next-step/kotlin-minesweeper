# kotlin-minesweeper



## 기능 요구사항
 - [ ] 높이와 너비, 지뢰 개수를 입력받을 수 있다.
    - [ ] 높이와 너비는 양수이어야한다.
 - [ ] 지뢰는 눈에 잘 띄는 것으로 표기한다.
    - [ ] 지뢰는 * 로 표기한다.
    - [ ] 지뢰가 아닌 것은 C 로 표기한다.
 - [ ] 지뢰는 가급적 랜덤에 가깝게 배치한다.
    - [ ] 지뢰 개수를 입력받는다.
    - [ ] 입력받은 지뢰 개수만큼 랜덤한 위치에 지뢰가 생성된다.
    - [ ] 지뢰 개수는 양수어야하며 (높이 * 너비) 개수보다 적어야 한다.
