## 🚀 1단계 - 지뢰 찾기(그리기)

### 요구사항 정리
- [x] 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- [x] 지뢰는 눈에 잘 띄는 것으로 표기한다.
- [x] 지뢰는 가급적 랜덤에 가깝게 배치한다.

10

너비를 입력하세요.

10
