# kotlin-minesweeper

---
### `1단계 - 지뢰찾기(그리기)`
<br>

##### `기능 요구사항`
- 지뢰 찾기를 변형한 프로그램을 구현
  - 높이와 너비, 지뢰 개수를 입력받을 수 있다.
  - 지뢰는 눈에 잘 띄는 것으로 표기한다.
  - "지뢰는 가급적 랜덤"에 가깝게 배치한다.

<br>

##### `실행결과`
```text
높이를 입력하세요.
10

너비를 입력하세요.
10

지뢰는 몇 개인가요?
10

지뢰찾기 게임 시작
C C C * C C C * C C 
C C * C * C C C C C
C C C C C C C C C C
C C C C C C C C C C
* C C C C C C C C C
C C C C C C * C C C
C C * C C C * C C C
C C C C C C * C C *
C C C C C C C C C C
C C C C C C C C C C
```

##### `요구사항 정리`
  - [x] 높이(height)와 너비(width)를 입력값은 1이상, 30이하의 정수이다
  - [x] 지뢰의 개수를 입력받는다
    - [x] 지뢰의 개수는 너비 * 높이 보다 많을 수 없다
  - [x] 지뢰찾기 맵은 높이 * 너비 만큼 생성된다
    - [x] 맵은 x축, y축 의 좌표(Coordinate)를 가진다
    - [x] 각 좌표는 지뢰(*) 또는 C 로 표시된다 
    - [x] 지뢰는 입력된 개수의 개수만큼 생성한다
    - [x] 지뢰의 생성은 랜덤이다
