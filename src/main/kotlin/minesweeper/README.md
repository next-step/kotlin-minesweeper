# 지뢰찾기
- 지뢰 찾기를 변형한 프로그램을 구현한다.
- 높이와 너비, 지뢰 개수를 입력받을 수 있다.
- 지뢰는 눈에 잘 띄는 것으로 표기한다.
- 지뢰는 가급적 랜덤에 가깝게 배치한다.

## 모델링

### MineField - 지뢰판
- 지뢰 찾기를 실행하기 위한 판을 나타낸다. 
- 필드들을 가진다. 
- 높이, 너비, 지뢰 갯수를 입력받아 지뢰 판을 생성한다. 

### Field - 필드
- 좌표와 지뢰유무를 나타내는 점을 가진다.

### Coordinate - 좌표 
- 지뢰 판에서의 위치를 나타내는 x,y 값을 가진다. 

### Dot - 지뢰 유무
- 지뢰 유무를 나타내는 값
